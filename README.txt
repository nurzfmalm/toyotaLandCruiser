TOYOTA LAND CRUISER - ЛЕГЕНДА БЕЗДОРОЖЬЯ
========================================

Автор: Касымов Нурзат 10К

ОПИСАНИЕ ПРОЕКТА
----------------
Профессиональный сайт о легендарном внедорожнике Toyota Land Cruiser.
Премиальный дизайн с темной цветовой схемой, золотыми и красными акцентами.

СТРУКТУРА САЙТА
---------------
- index.html - Главная страница
- history.html - История Land Cruiser с 1951 года
- models.html - Все поколения
- specs.html - Технические характеристики и таблицы
- gallery.html - Галерея
- styles.css - Премиальные стили
- README.txt - Этот файл
- assets/ - Медиафайлы

ИСПОЛЬЗОВАННЫЕ ТЕХНОЛОГИИ
-------------------------
HTML5:
- <!DOCTYPE html>
- Семантические теги: header, nav, main, section, article, aside, footer
- Метатеги: charset, viewport, description, author
- <time datetime="">
- <h1> - <h3>
- <p>, <strong>, <em>, <mark>
- <ul>, <ol>, <li>
- <dl>, <dt>, <dd>
- <a href=""> (внутренние и внешние)
- <a href="#id"> (якорные ссылки)
- <table>, <caption>, <thead>, <tbody>, <tfoot>, <tr>, <th scope>, <td>
- <blockquote>, <cite>
- <details>, <summary>
- <audio>, <source>
- <div> с классами
- <small>

CSS3:
- Внешний файл styles.css
- CSS Variables (--primary-dark, --accent-gold, и др.)
- Flexbox для навигации
- CSS Grid для:
  * Сетки карточек (features-grid)
  * Таймлайн (timeline-item)
  * Таблицы (tables-grid)
  * Футер (footer-content)
- Градиенты (linear-gradient)
- Transitions и hover эффекты
- Box-shadow для глубины
- Media queries для адаптивности
- Псевдоэлементы (::before, ::after)
